## ğŸš€ Quick Start

### Prerequisites
- Node.js 18+ installed
- MongoDB Atlas account (or local MongoDB)
- OpenAI API key

### Installation

1. **Clone and Install**
   ```bash
   npm install
   ```

2. **Configure Environment**
   ```bash
   # Copy example env file
   cp .env.local.example .env.local
   
   # Edit .env.local and add your OpenAI API key
   OPENAI_API_KEY=sk-your-key-here
   ```
   
   Get your OpenAI key: https://platform.openai.com/api-keys

3. **Start Development Server**
   ```bash
   npm run dev
   ```

4. **Run Tests**
   ```bash
   node test-ai-system.js
   ```

5. **Open Application**
   ```
   http://localhost:3000
   ```

### Alternative Setup (Windows)
```bash
setup-ai-workspace.bat
```

### Alternative Setup (Linux/Mac)
```bash
chmod +x setup-ai-workspace.sh
./setup-ai-workspace.sh
```
