## Animation Examples

### Entry Animations
```
Header:    fade-in + slide-in-from-top
Sidebar:   fade-in + slide-in-from-left
Content:   fade-in + slide-in-from-right
Stats:     Staggered (100ms delay each)
```

### Hover Animations
```
Cards:     scale-105 (5% larger)
Buttons:   scale-[1.02] (2% larger)
Avatar:    scale-105 + shadow increase
Stats:     scale-105 + shadow glow
```

### Interaction Animations
```
Upload:    Spinner during upload
Error:     Slide in from top
Success:   Fade in
Dismiss:   Fade out
```
